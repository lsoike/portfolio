DATA VISUALIZATION

CHALLENGE

In this assignment you should create a unique data visualization using a small set of data. Your program must dynamically create the visualization from the data set (this means I should be able to change your data, and the visualization will change too). Your visualization should be graphically interesting, and should include some sort of user interaction or changes based on actions the user takes.

DESCRIPTION

This visualization shows my heart rate during each minute of a 30-minute workout. Data was collected from my Apple Watch. As you hover over each data point, the heart rate and minute of the workout are displayed.
