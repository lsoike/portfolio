CHALLENGE

You will program your very own Lego family. You can base it off a TV show, a movie, a comic, or even your own family if you want. Check Imagine by Lego for more inspiration. This assignment will help you practice with functions, variables, declarations and assignment statements. These ideas help make computations dynamic in interesting ways.

DESCRIPTION

I created a Lego family representation of the Jetsons. This program shows each character starting in a different corner, then moving to all meet in the middle.
